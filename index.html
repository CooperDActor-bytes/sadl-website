Welcome to Spearmint

Your favorite open-source Peppermint installation companion!



Install Spearmint

root@spearmint:~# curl -sSL -o deployer.sh https://deploy.spearmint.sh && chmod +x deployer.sh && ./deployer.sh


Video Showcase



Key Features

Easy to Use

Spearmint does almost everything for you, making your Peppermint installation process easier!

Secure

Spearmint uses a secure download system, preventing anything from happening in the event of a domain issue!

Lightweight

Spearmint is build to be lightweight, so it can be used on almost anything (Except a potato or toaster)



Compatibility

Operating System Compatibility Notes
Ubuntu Tested on v20, v22 & v24
Debian Tested on v11 & v12
CentOS Compatibility coming in v3.1
Fedora Compatibility coming in v3.1
FreeBSD Spearmint will not be made compatible with this OS

We are always working on becoming compatible with more server operating systems!
